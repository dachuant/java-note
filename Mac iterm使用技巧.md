# 1. Mac iterm使用技巧
```
command + D 纵向分割
command + shift + D 横向分割
command + W 关闭当前窗口
command + [] 切换窗口
command + 上下左右切换当前窗口子窗口

command + T 新开tab
command + 左右方向键切换tab
command + 数字切换第几个tab
command + shift + []切换tab

command + shift + enter 最大化还原窗口
command + enter 最大化还原窗口
command + /快速定位光标

command + alt + E 搜索tab，快速定位
command + M 设置标记
command + J跳转上个标记
command + shift + 上下键切换标记
command + alt +；鼠标所在行高亮显示
```
